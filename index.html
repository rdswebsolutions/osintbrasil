❌ Introdução

A coleta de dados através de fontes abertas (OSINT) evoluiu com técnicas legais, acessíveis e eficazes para monitoramento. Esta cartilha alerta sobre o uso de serviços de publicidade de operadoras móveis para rastreamento e engenharia social.

📶 O que está acontecendo?

  • Sites acessados via internet móvel
  • Ligações efetuadas e recebidas
  • Presença em regiões geográficas (geozonas)

Esses dados são vendidos como “inteligência de mercado”, mas podem ser usados para vigilância.

⚡ Técnicas Avançadas (OSINT com esteroides)

  • Criação de geozonas para rastreamento
  • Disparo de SMS via eventos geográficos
  • Uso de chamadas rastreáveis para ativar filtros
